Todos los formularios que se manejan en la web utilizan metodos para el envio o respuesta de la informacion, los mas importantes son:


Get: Este metodo se encarga de enviar la informacion por la url o salida estandar del navegador, esto propicia que se pueda visualizar toda la informacion del formulario
post:Este metodo se encarga de enviar la informacion por la url pero de forma oculta lo que ofrece cierto nivel de seguridad, este metodo se utiliza comunmente para insetar un nuevo registro dentro del sistema.
